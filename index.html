HTML Generator

Simple & Fast HTML Template Generator in Bash Shell

View on GitHub

✨ Features

  • ✅ Generate HTML templates interactively
  • 🎨 Supports Bootstrap 5, Tailwind, or Vanilla CSS
  • 📝 Customizable meta tags (title, description, author)
  • 💻 Open Source and lightweight Shell script
  • 🚀 Includes Debian package with launcher & icon

🔧 Installation

Install using the Debian package:

sudo dpkg -i html-generator_1.0.0.deb

Or clone the repository:

git clone https://github.com/jancordeiro/html-generator.git
cd html-generator
chmod +x html-generator.sh
./html-generator.sh

💡 Usage

Simply run:

html-generator

And follow the interactive prompts.
